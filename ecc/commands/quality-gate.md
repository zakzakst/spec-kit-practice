# 品質ゲートコマンド

ECC の品質パイプラインを、ファイル単位またはプロジェクト単位で必要時に実行します。

## 使い方

`/quality-gate [path|.] [--fix] [--strict]`

- デフォルト対象: 現在のディレクトリ（`.`）
- `--fix`: 設定されている範囲で自動整形 / 自動修正を許可する
- `--strict`: 対応しているツールでは warning も失敗扱いにする

## パイプライン

1. 対象の言語 / ツール構成を検出する。
2. フォーマッタのチェックを実行する。
3. 利用可能なら lint / type check を実行する。
4. 簡潔な改善リストを出す。

## メモ

このコマンドはフックの動作を手動実行できるようにしたものです。

## 引数

$ARGUMENTS:

- `[path|.]` 任意の対象パス
- `--fix` 任意
- `--strict` 任意
