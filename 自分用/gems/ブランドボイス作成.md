# 依頼：ブランドボイス・ガイドラインの作成 アップロードしたソース（ブランドコンセプト資料）の内容を深く分析し、このサービス専用の「ブランドボイス・ガイドライン」を作成してください。

# 出力構成案
1. ブランドボイスの定義（人格）： ブランドを一人の人間に例えた時の性格や話し方を、ソースの価値観に基づいて定義してください。
2. ボイスチャート（Do/Don'tリスト）： 実務で迷わないよう、以下の3〜4つの特性について、「意味」「推奨する表現（Do）」「避けるべき表現（Don't）」をまとめた表を作成してください。
3. 場面別のトーン調整ルール： 以下の各媒体において、どのように「声の調子」を調整すべきか具体的に指示してください。
  - 営業用プレゼン資料（信頼感・専門性）
  - SNS投稿（親しみやすさ・会話体）
  - プレスリリース（誠実・客観的）
  - 自社ブログ（共感・丁寧な解説）
4. AI執筆用のライティング・ルール： AI（生成AI）にこのブランドらしい文章を書かせるための、具体的な文体ルールを指定してください（例：能動態の使用、平易な日常語、専門用語の禁止など）。
5. NGワード・推奨表現リスト： ブランドの品位を損なう言葉や、好んで使うべきキーワードを列挙してください。
