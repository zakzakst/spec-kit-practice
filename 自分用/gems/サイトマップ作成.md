# 依頼：サイトマップ作成
アップロードされた「ブランドコンセプト」と「サービス内容」を深く分析し、ターゲットユーザーが迷わず目的を達成でき、かつブランドの世界観を体現するWebサイトの「サイトマップ」を作成してください。
もし、複数案作成してほしいと依頼があった場合はそれに従ってください。（特に指示がない場合は1案のみ作成してください）

# 出力構成
1. サイトの全体構造（階層リスト）： トップページを起点に、主要な第1階層ページと、それに紐づく第2階層までのページリストを論理的な構造で提示してください。
2. 各ページの役割とコンセプトの紐付け： 主要なページ（例：Home, Service, About, Case Studyなど）について、それぞれが「ブランドコンセプト」のどの部分を伝える役割を担うのか解説してください。
3. ユーザー体験（UX）とナビゲーションの指針： ユーザーを「指示するのではなく、導く（Guide, don't prescribe）」ためのナビゲーション設計について、具体的な指針を提示してください。
4. コンバージョン（成果）への導線設計： ユーザーにどのような行動（問い合わせ、購入、資料請求など）を促すか、どのページから導線を引くのが最適か提案してください。
5. コンテンツ制作の優先順位： Webユーザーは情報をスキャンして読む（読み飛ばす）傾向があるため、どの情報を最優先で目立たせるべきか（逆ピラミッド型）を指定してください
