# 依頼：VI（ビジュアル・アイデンティティ）マニュアルの作成
アップロードしたブランドコンセプト資料を分析し、その世界観を視覚的に統一するための「VIマニュアル」を作成してください。
もし、複数案作成してほしいと依頼があった場合はそれに従ってください。（特に指示がない場合は1案のみ作成してください）

# 出力構成
1. デザインコンセプトの言語化： ブランドコンセプトから導き出される「視覚的な印象（トーン＆マナー）」を言語化してください（例：Appleの『シンプルで革新的』、無印良品の『引き算の美学』のような指針）。
2. ロゴの使用ルール（Do/Don't）： ブランドの品位を守るため、以下の項目についてルールを提案してください。
  - アイソレーション（保護空間）の確保
  - 最小使用サイズの規定
  - 禁止事項（色の変更、変形、重ね方など）
3. カラーパレットの定義： コンセプトの情緒的価値を表現する、メインカラー、サブカラー、アクセントカラーを選定・提案してください。
4. タイポグラフィ（書体）： ブランドの人格を表現するのに最適な、見出し用および本文用のフォントの方向性を指定してください。
5. イメージ・イラスト・写真のトーン： 使用する写真やイラストが備えるべき「空気感」や「構図」のルールを定義してください（例：自然光を活かす、余白を広く取るなど）。
6. 媒体別の展開イメージ： 以下の媒体で、どのようにデザインを統一すべきか指針を示してください。
  - Webサイト（操作性と一貫性）
  - パンフレット・資料（レイアウトと視認性）
  - 製品・パッケージ（開封体験や素材感）
